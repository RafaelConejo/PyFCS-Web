# PyFCS-Web
Web-based Fuzzy Color Spaces Framework

PyFCS-Web is the web version of **PyFCS**, a framework for creating, visualizing,
and applying **fuzzy color spaces**.  
It brings fuzzy color analysis to the browser, offering an interactive and
visual environment to work with colors and images.


## Main Features

### Create Fuzzy Color Spaces
- Build new fuzzy color spaces from predefined color palettes
- Add new colors manually or by sampling pixels from images
- Name and export custom fuzzy color spaces

### Visualize Color Spaces
- Interactive 3D visualization in CIELAB space
- Rotate, zoom, and explore fuzzy color distributions
- Enable or disable individual color components

### Image Management
- Load images from your computer or use preset images
- Each image opens in its own movable and resizable window
- Work with multiple images simultaneously

### Color Mapping
- Visualize the membership of a single fuzzy color in an image
- Assign each pixel to its most representative fuzzy color
- Compare different color representations interactively

### Color Picking from Images
- Pick colors directly by clicking on images
- Automatic conversion to LAB color space
- Add sampled colors to palette-based color space creation


## Goal

The goal of PyFCS-Web is to make fuzzy color spaces easier to **create, explore,
and apply**, providing a modern web interface that removes the need for
desktop-based tools.


## Based On

PyFCS-Web is based on the original **PyFCS** Python framework and extends it with
a fully interactive web interface.
